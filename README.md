EcoTrack Client (Firebase)

Purpose

1. Modern, responsive React app for eco-friendly challenges and activity tracking with Firebase Authentication.

Live URL

1. My deployment URL here: `https://eco-track-10.netlify.app/`

Key Features

1. Email/password and Google sign-in
2. Password reset via email
3. Profile: view user info, edit display name and photo URL
4. Challenges: browse, join, update, delete
5. Activities: bylogged-inuserlist
6. Toast-based feedback and clean Tailwind + DaisyUI UI

Tech Stack

1. React 19, React Router 7, Vite
2. Tailwind CSS v4 + DaisyUI
3. Firebase Authentication
4. lucide-react, react-toastify

Setup

1. Prerequisites: Node.js 18+, backend API at `https://eco-track-server-nine.vercel.app/`
2. Install: `npm install`
3. Dev: `npm run dev`
4. Build: `npm run build` | Preview: `npm run preview`
5. Lint: `npm run lint`

Configuration

1. Firebase: `src/Firebase/firebaseConfig.js`
2. Router: `src/main.jsx`
3. Styles: `src/index.css`
